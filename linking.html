Links in HTML

In this lesson, we examine absolute and relative URLs, and the target attribute for linking in HTML.

Absolute and Relative URLs

This is an absolute URL.

It will always open the About Us page on www.example.com no matter where the link is placed.

This is a relative URL.

It uses the location of textbasics.html relative to the page you're currently viewing to direct the user to that page.

It's best to use relative links as they save time on maintenance.

For instance, were the website owner to change their domain, they would need to replace all instances of "https://example.com/about.html" on all pages with the corresponding URL in the new domain. However, clicking the "textbasics.html" link will always redirect to textbasics.html no matter the domain name.

The "target" attribute

The target attribute specifies where to open the linked webpage.

By default, links wil open in the same tab you're currently on.

The "_blank" target changes this behavior and opens the link in a new tab.
