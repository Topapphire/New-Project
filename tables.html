Tables in HTML

This lesson contains basic table syntax for organizing data in rows and columns. It also explains the concepts of scope and cell merging with HTML.

Creating Simple Tables

Create tables using the <table> tag. <tr> creates table rows, <td> refers to a cell, and <th> to table header.

Specifying Scope

The scope attribute specifies which row(s) or column(s) an header applies to.

Scopes apply to the column (scope="col") by deafult, but you may change the behavior using the syntax scope="row".

Merging Cells

You can merge cells across rows or columns in HTML. Use either the rowspan (to merge cells vertically across rows) or colspan (to merge cells horizontally across columns) attributes depending on your goal.

Making Tables Accessible

It is a best practice to add captions to tables, use the scope attribute with headers, and use semantic markup correctly so screen readers know how to handle your tables.
