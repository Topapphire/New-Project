Audio & Video

This lesson explains how to embed audio and video files in your webpage, use different sources, and add accessibility features to media files in HTML.

Embedding Audio and Video

Use the <audio> and <video> elements to include audio and video content on your web page. It's a best practice to add width and height attributes to specify the video's size. Add the <controls> attributes to display the browser's native playback controls.

Using the <source> Element

The <source> element allows you to specify multiple media sources for the same audio or video, so the browser can choose the most compatible format for it.
