Images in HTML

This module focuses on using images in HTML. The focus is on adding images to webpages, using alt text for accessibility, and <figure> and <figcaption> elements.

A typical image element is created like this <img src="path/to/image.jpg" alt="Description of the image">

Alt Text, <figure>, and <figcaption> Attributes

Alt text aids accessibility by providing a description of the image. The <figure> and <figcaption> attributes semantically group media content with its description.
